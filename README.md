灵键拼音（LingJian Pinyin）
===========================

灵键拼音是一个基于 Fcitx5 的中文拼音输入法项目，目标是提供 **轻量、响应快、可高度定制** 的输入体验。

项目当前处于原型/实验阶段，主要关注：

- 拆分「输入引擎」与「UI 显示」两部分，方便后期替换或扩展界面
- 在 Fcitx5 中嵌入输入引擎，UI 可以作为独立进程，通过 IPC 通信
- 为后续加入智能联想、自定义短语、脚本扩展等能力预留接口

## 目录结构（规划）

- `src/fcitx/`：Fcitx5 插件入口与引擎适配层
- `src/core/`：输入上下文、拼音解码、候选生成等核心逻辑
- `src/dict/`：系统词典、用户词典实现
- `src/ui/`：独立 UI 进程（候选框、状态栏等，推荐使用 Qt 实现）
- `src/utils/`：通用工具
- `data/`：词库、语言模型等数据文件

随着开发推进，具体结构可能会有所调整。

## 构建方式（示例）

项目计划使用 CMake 作为构建系统，假设你已经安装好 Fcitx5 与（可选的）Qt 等开发依赖，可以类似这样构建：

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

当前代码更偏向于框架与接口形状的设计，后续会逐步补充真实的输入引擎逻辑与更完整的 UI 实现。
